# Security Policy

## Supported Versions
EWIS is under active development. Security fixes are provided for the most recent minor release line.

## Reporting a Vulnerability
Please do not open a public issue for security vulnerabilities.

Email: aristocles24@gmail.com  
Subject: [EWIS SECURITY] <short summary>

Include:
- impacted versions
- reproduction steps
- potential impact
- any relevant logs or proofs

We will acknowledge receipt and coordinate a fix and disclosure timeline.
